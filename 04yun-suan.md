# 运算符

## 数学运算

```
+-*/%
```

```
var x = 1, y = 1;
x = y++ // x = 1, y = 2
x = ++y // x = 3, y = 3
var z = "1"
x + z // "31",数字和字符串相加结果为数字
```

## 逻辑运算

```
&& || !
```

## 位运算

- 按位与 &

```
1 & 2 // 01 & 10 => 0
1 & 3 // 01 & 11 => 1
if(n & 1)
{
  console.log("n是基数")
}
```

- 按位或 |

```
1 || 2 // 01 | 10 => 3
1 || 4 // 01 | 100 => 5

// 浮点型向下取整
var num = 1.1 | 0
```

- 按位非 ~

```
var num = 1
var num1 = ~num
//  num: 00000000000000000000000000001
// ~num: 11111111111111111111111111110
// js中默认数字是有符号的，所以取非之后的数字为负数
// 负数二进制转化为十进制：最高位符号位不变，其他位取反，然后+1，
```


```
& | ~ ^ << >>
```
