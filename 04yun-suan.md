# 运算符

## 数学运算

```
+-*/%
```

```
var x = 1, y = 1;
x = y++ // x = 1, y = 2
x = ++y // x = 3, y = 3
var z = "1"
x + z // "31",数字和字符串相加结果为数字
```

## 逻辑运算

```
&& || !
```

## 位运算

- 按位与 &

```
1 & 2 // 01 & 10 => 0
1 & 3 // 01 & 11 => 1
if(n & 1)
{
  console.log("n是基数")
}
```

- 按位或 |

```
1 || 2 // 01 | 10 => 3
1 || 4 // 01 | 100 => 5
// 浮点型向下取整
// 浮点型向上取整
```


```
& | ~ ^ << >>
```
