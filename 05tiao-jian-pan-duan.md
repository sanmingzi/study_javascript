## 逻辑运算符

```
&&
||

```

## 三目运算

```
condition ? value1 : value2;
```

## if...else

```
if(condition1) {

}else if(condition2) {

}else {

}
```

## switch

```
var d = new Date().getDay();
switch(d) {
  case 6:
    console.log("今天是星期六");
  break;
  case 0:
    console.log("今天是星期日");
  break;
  default:
    console.log("今天不是周末");
  break; 
}
```
